# `aren setup`

Initialize `~/.aren/aren.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
aren setup
aren setup --workspace ~/.aren/workspace
```

To run the wizard via setup:

```bash
aren setup --wizard
```

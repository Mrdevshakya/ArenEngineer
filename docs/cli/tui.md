# `aren tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/tui)

## Examples

```bash
aren tui
aren tui --url ws://127.0.0.1:18789 --token <token>
aren tui --session main --deliver
```
